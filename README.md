Requirements
============

* [Docker](https://www.docker.com/)

How to install?
===============

```
$ git clone --recursive git@github.com:netenberg/fantastico_f3_docker_7.0.git
```

How to use?
===========

```
$ make clean
$ make build
$ FANTASTICO_F3_CORE=/var/netenberg/fantastico_f3/sources make run
```
